# 🌟 小恋的留言板

> 一个用 Streamlit 写的迷你留言板~
> 写着玩的小玩意,欢迎来玩哦！

## 🎈 这是什么?

就是一个简单的留言板啦,可以:
- 📝 写写留言
- 🖼️ 放图片
- 📹 传视频
- 🎵 传音频
- ✨ 互相交流

## 🔑 管理员功能

## 📁 文件放哪了?

```
message/
├── app.py           # 主程序
├── messages.json    # 留言数据
└── uploads/         # 上传的文件
    ├── images/      # 图片
    ├── videos/      # 视频
    └── audios/      # 音频
```
